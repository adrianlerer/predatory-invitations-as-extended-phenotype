## Pattern Progression

The cases show **evolutionary diversification** of PSO tactics:

1. **Basic automation** (template errors)
2. **False follow-ups** (psychological manipulation)
3. **Catch-all scopes** (maximize capture)
4. **Explicit pricing** (no pretense of review)
5. **Longer deadlines** (mimic legitimacy)

Each case file includes:
- Full email content (anonymized)
- Pattern analysis
- PSO score breakdown
- Extended Phenotype interpretation
- Comparison to other cases
